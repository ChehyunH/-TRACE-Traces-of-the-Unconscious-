# 🧠 TRACE: Traces of the Unconscious
> **심박으로 기록되는, 보이지 않던 시간**

---

## 1. 🎯 기획 의도: 왜 '무의식'인가?
우리는 흔히 **'시간이 빠르다'**는 말을 합니다. 어쩌면 우리가 의식하지 못한 채 지나쳐버린 순간들이 많기에 시간이 더 빠르게 느껴지는 것은 아닐까요? 

주변의 수많은 자극 속에서 우리는 자극을 '느끼기'보다 '통과'시키며 살아갑니다. **TRACE**는 우리가 무의식적으로 흘려보낸 그 소중한 시간들을 시각적 흔적으로 기록하기 위해 시작되었습니다.

## 2. 💓 데이터의 선택: 심박수(Heart Rate)
무의식을 기록할 가장 정직한 데이터를 고민했습니다.
* **초기 구상:** 자극에 대한 반응을 가장 즉각적으로 보여주는 **뇌파(EEG)**를 고려했습니다.
* **최종 선택:** 일상적인 접근성을 위해 **스마트워치의 심박수 데이터**를 활용했습니다. 
* **이유:** 심박수는 내가 의식적으로 통제할 수 없는 생체 신호이며, 긴장, 몰입, 평온 등 나의 상태를 가장 꾸준하고 정직하게 반영하는 '무의식의 언어'이기 때문입니다.

## 3. ✨ 시스템의 핵심
* **일상의 시각화:** 하루 종일 측정된 심박 변화를 예술적인 생성 예술(Generative Art)로 변환합니다.
* **기록의 아카이빙:** 10분마다 저장된 스냅샷은 측정이 종료시 영상으로 변환됩니다. 사용자는 이를 통해 하루를 되짚어볼 '시간적 여유'를 선물 받습니다.
* **개인화된 화풍:** 사용자의 취향과 그날의 기분에 따라 화가들의 화풍을 선택하여 나만이 작품이 완성됩니다.

## 4. 🎨 예술적 문법 (Artistic Style)
심박 데이터는 아래 세 가지 화풍의 추상 문법으로 번역됩니다.

| 작가 | 시각적 특징 | 데이터의 해석 |
| :--- | :--- | :--- |
| **GOGH** | 거친 붓질과 소용돌이 | 강렬한 자극과 감정의 동요를 선형적 에너지로 표현 |
| **MONET** | 부드러운 블롭과 잔상 | 시간의 흐름 속에 머무는 고요하고 몽환적인 상태 구현 |
| **PICASSO** | 폴리곤과 분절된 형태 | 자극에 의해 조각난 인식의 파편들을 기하학적으로 재구성 |

## ⚙️ 기술 파이프라인 (The Mechanism)
1. **Garmin Data:** 가민 스마트워치를 통한 실시간 심박수 수집.
2. **Baseline Learning:** 개인별 심박 변동폭 학습 및 정규화(Normalization).
3. **Generative Flow:** 2초 간격 샘플링을 통한 실시간 디지털 캔버스 생성.
4. **Summary Card:** 10분 단위의 흔적을 개별 카드로 기록 (시간의 증거).
5. **Final Trace:** **시간 감쇠(Time Decay)** 기법 적용. 오래된 흔적은 배경으로 깊게 가라앉고, 의미 있는 변화들만 층층이 쌓여 시간이 길어질수록 깊이감 있는 이미지 완성.

## 🌈 최종 가치: 회상, 그리고 다시 사랑하기
하루를 마무리하며 완성된 그림을 마주하는 순간, 사용자는 깨닫게 됩니다.
> *"빠르게 지나가 버린 줄 알았던 나의 하루가 사실은 수많은 흔적으로 가득 차 있었구나."*

TRACE는 데이터를 분석하는 도구가 아닙니다. 사용자가 여유를 갖고 자신의 하루를 돌아보며 **위로**를 얻고, 무의식 속에 흐르던 **시간의 가치**를 다시 발견하여 오늘 하루 감사하게 만드는 장치입니다.
